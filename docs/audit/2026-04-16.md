### 3.3 발견된 이슈

#### [P0] 🔴 Next.js 16 마이그레이션 경고

```
⚠ The "middleware" file convention is deprecated.
  Please use "proxy" instead.
```

- **영향**: Next.js 16.x에서 `middleware.ts` deprecated, 향후 제거 예정
- **조치**: `middleware.ts` → `proxy.ts`로 rename + API 시그니처 확인 필요
- **참고**: https://nextjs.org/docs/messages/middleware-to-proxy

#### [P1] 🟡 Dev 서버에서 API Route Turbopack 컴파일 hang

- **증상**: `/api/intelligence/value-hunter` 등 API 요청이 90초 타임아웃
- **서버 로그**: API 요청 자체가 로그에 찍히지 않음
- **프로덕션 빌드**: 정상 (빌드 자체는 성공)
- **가설**:
  1. Next.js 16 Turbopack + middleware 조합 이슈
  2. middleware.ts의 Supabase SSR 초기화 실패
  3. 특정 API route의 import chain 문제
- **조치 필요**: 프로덕션 서버(`npm run start`)에서 재검증 (현재 포트 3000 좀비 프로세스로 차단됨)

#### [P2] 🟢 MEMORY.md 불일치

- `/etf-signals`, `/relay`가 MEMORY.md에 독립 페이지로 기재돼 있으나 실제로는 `/quant`로 리다이렉트
- 업데이트 시점: 리다이렉트 변경된 커밋 확인 필요
