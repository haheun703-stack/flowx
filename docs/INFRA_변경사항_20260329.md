## 3. Supabase 새 테이블

### `kospi_investor_daily`

| 컬럼 | 타입 | 설명 |
|------|------|------|
| date | date (PK) | 거래일 |
| foreign_net | bigint | 외국인 순매수 (백만원) |
| inst_net | bigint | 기관 순매수 (백만원) |
| indiv_net | bigint | 개인 순매수 (백만원) |
| created_at | timestamptz | 생성 시각 |

### 데이터 입력 방법
- **자동**: 웹봇 cron (`/api/cron/update-market`)이 매 실행마다 upsert
- KIS API `fetchInvestorTrend()` 결과를 그날 날짜로 저장
- 봇에서 직접 이 테이블에 쓸 필요 없음

### 읽기 API
```
GET https://www.flowx.kr/api/market/investor-flow
→ 최근 30일 투자자별 순매수 배열 반환
```
